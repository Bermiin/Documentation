Here you can find Example Products's PlaceholderAPI placeholder and in-built placeholders.

## PlaceholderAPI

These are PlaceholderAPI placeholders and can be used in messages, menus, etc.
They also let you display information in other plugins.

Placeholder            | Returns                   | Example
---------------------- | ------------------------- | ------------------
`%exampleplugin_name%` | The example plugins name. | `ExamplePlugin123`

## Included

Placeholder            | Returns                   | Example
---------------------- | ------------------------- | ------------------
`<prefix>` | The plugin prefix for messages. | `[ExamplePlugin]`
