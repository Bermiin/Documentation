### Information

MongoDB is not recommended for Apollo.

### Hosting

[MongoDB Atlas](https://www.mongodb.com/atlas/database) has free small MongoDB databases.

### Configuration

Field          | Example     | Description
-------------- | ----------- | -------------------------------------------
`host`         | `127.0.0.1` | The database hostname.
`port`         | `27017`     | The port the database is running on.
`storage_type` | `MONGODB`   | The database type.
`username`     | `root`      | The username to login to the database with.
`password`     | `pass`      | The password to login to the database with.
`name`         | `apollo`    | The database name.
