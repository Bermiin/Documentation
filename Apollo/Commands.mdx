A list of all of Apollo's commands.

:::info
If you use Hider, replace `apollo` with your custom command name.
:::

`<>` = Required `[]` = Optional

Command                               | Permission                          | Description
------------------------------------- | ----------------------------------- | ---------------------------------------------
`/apollo`                             | `apollo.use`                        | Displays the help menu.
`/apollo alerts`                      | `apollo.use`, `apollo.alerts`       | Toggles anticheat alerts.
`/apollo reload`                      | `apollo.use`                        | Reloads the configuration.
`/apollo exempt [player]`             | `apollo.use`, `apollo.toggleexempt` | Toggle a player's exempt status.
`/apollo info <player>`               | `apollo.use`                        | View information about an online player.
`/apollo logs <player>`               | `apollo.use`                        | Print a player's anticheat logs.
`/apollo recentlogs <player>`         | `apollo.use`                        | Print a player's recent anticheat logs.
`/apollo pastelogs <player>`          | `apollo.use`                        | Creates a paste of a player's logs.
`/apollo punishments player <player>` | `apollo.use`                        | Displays a player's punishments.
`/apollo punishments id <banid>`      | `apollo.use`                        | Displays info about a punishment.
