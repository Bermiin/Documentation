Apollo is a highly configurable, Minecraft Anticheat that provides an advanced solution for removing cheaters your server. Apollo has numerous checks, an analysis engine, advanced database support, machine learning features, a powerful punishment and violation system, an advanced threading system, prediction systems and more.

## Installation

<Tabs queryString="type">
    <TabItem value="spigot" label="Spigot / Paper" default>
        1. Download and install Retention, which can be found on the resource page, into your plugins folder.<br/>
        2. Start your server and wait until it finishes booting up, once finished, stop your server.<br/>
        3. Open loader.yml in Retention's plugin folder and enable Apollo, save and reboot.<br/>
    </TabItem>
</Tabs>

## Setup Guide

1. Setup the [Database](/Apollo/Features/Database).
2. Enable [Realtime Network Sync](/Apollo/Features/NetworkSync).
3. Configure [Punishments](/Apollo/Features/CheckGroups).
