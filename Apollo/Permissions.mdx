| Permission Node                           | Use                                                                                     |
| ------------------------------------------|---------------------------------------------------------------------------------------- |
| `apollo.alerts`                           | Get alerts from the anticheat                                                           |
| `apollo.use`                              | Access to the `/apollo` command                                                         |
| `apollo.falses`                           | Use the report falses button. This button should be used with extreme care.             |
| `apollo.bypass`                           | Bypass the anticheat.                                                                   |
| `apollo.switchserver`                     | Switch servers using the buttons in the Redis Alerts.                                   |
| `apollo.vpn.exempt`                       | Bypass the kick on vpn feature, if it is enabled in the config.                         |
| ------------------------------------------|---------------------------------------------------------------------------------------- |
