# FAQ

<details>
<summary>How can I set up ranks?</summary>

You can set up ranks by using the `/rank` command. Be sure to add yourself to security (/security add yourname). You can create a rank by doing `/rank create` and then editing it in `/rank editor` or you can use the provided commands to edit it manually by doing `/rank [command]`
</details>

<details>
<summary>How can I set up reports?</summary>
 
You can set up reports by using the `/adminreport` command. Every report category is linked to a punishment ladder. Those can be setup with `/pladder create id requiresChatSnapshot (true/false) reason`. Be sure to also add punishments to the ladder with `/pladder addpunishment`. To create a category type `/adminreport create id punishmentLadder material displayName`.
</details>

<details>
<summary>My menus are not working on 1.12+ servers. How do I fix?</summary>

To ensure your menus work properly on 1.12+ servers, follow these steps:

1. Go to menus.yml file.
2. Locate the "fill-glass" section.
3. Change the material to something like "GRAY_STAINED_GLASS_PANE" under "fill-glass".
4. Save and restart your server.

The menus should start working properly for 1.12+ servers again.
</details>

<summary>Why am I being blacklisted randomly when I op someone or I get opped?</summary>

This is because of our security system we added. Please do `/security add [your username]` or disable it in the config.
</details>

<details>
<summary>How do I grant a rank from console?</summary>

You can grant a rank from the console using the `/ogrant` command
</details>

<details>
<summary>How do I add a tag to a rank?</summary>

Add the permission `core.tags.[tagName]` (or `core.tags.*` if you wish to add all tags) to the rank
</details>
