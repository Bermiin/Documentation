A list of permissions for bolt and all its features.

If you use Phoenix, please take a look at [Phoenix/Features/Ranks/Permissions](/Phoenix/Features/Ranks#permissions) for detailed information on permissions and how you can set them up.

:::warning
Bolt feature pages are coming soon, disregard the link to them above the permissions.
:::

## Kits

Visit [Features/Kits](/Bolt/Features/Kits) for more information regarding kit permissions.

Permission       | Description
---------------- | -----------
`bolt.kit.admin` | Access to kit management commands.

## Arenas

Visit [Features/Arenas](/Bolt/Features/Arenas) for more information regarding arena permissions.

Permission         | Description
------------------ | -----------
`bolt.arena.admin` | Access to arena management commands.
`bolt.arena.setup` | Access to arena setup commands.

## Events

Visit [Features/Events](/Bolt/Features/Events) for more information regarding event permissions.

Permission         | Description
------------------ | -----------
`bolt.event.admin` | Access to event admin commands.

## Tournaments

Visit [Features/Tournaments](/Bolt/Features/Tournaments) for more information regarding tournament permissions.

Permission              | Description
----------------------- | -----------
`bolt.tournament.admin` | Access to tournament admin commands.
`bolt.tournament.host`  | Access to hosting tournaments.

## Cosmetics

Visit [Features/Cosmetics](/Bolt/Features/Cosmetics) for more information regarding cosmetic permissions.

Permission                 | Description
-------------------------- | -----------
`bolt.killeffect.*`        | Access to all kill effects.
`bolt.killeffect.<name>`   | Access to a specific kill effect.
`bolt.killmessages.*`      | Access to all kill messages.
`bolt.killmessages.<name>` | Access to specific kill messages.
`bolt.trails.*`            | Access to all trails.
`bolt.trails.<name>`       | Access to specific trails.

## Queues

Visit [Features/Queues](/Bolt/Features/Queues) for more information regarding queue permissions.

Permission         | Description
------------------ | -----------
`bolt.queue.admin` | Access to queue admin commands.

## Bots

Visit [Features/Bots](/Bolt/Features/Bots) for more information regarding bot permissions.

Permission           | Description
-------------------- | -----------
`bolt.bot.custom`    | Allows you to play a custom bot preset.
`bolt.bot.admin`     | Access to bot admin commands.
`bolt.bot.knockback` | Access to set bot knockback.

## Parties

Visit [Features/Parties](/Bolt/Features/Parties) for more information regarding party permissions.

Permission              | Description
----------------------- | -----------
`bolt.party.limit`      | Bypass the maximum default party limit.
`bolt.party.public`     | Allows toggling of the party's privacy.
`bolt.party.announce`   | Alerts players about your open party.
`bolt.party.pingfactor` | Access to change the party queue ping range.

## Profiles

Visit [Features/Profiles](/Bolt/Features/Profiles) for more information regarding profile permissions.

Permission                    | Description
----------------------------- | -----------
`bolt.profile.revert`         | Allows you to revert matches and elo.
`bolt.profile.ranked`         | Bypasses the minumum unranked wins requirement.
`bolt.profile.ping`           | Bypasses the maximum ping requirement for ranked.
`bolt.profile.arena`          | Access to pick the arena you want to play on.
`bolt.profile.fly`            | Allows you to fly in the lobby.
`bolt.profile.flymode`        | Allows toggling of post match flight.
`bolt.profile.dropitems`      | Allows toggling of opponent item drops.
`bolt.profile.clearinventory` | Allows toggleing of post match inventory clear.
`bolt.profile.deathanimation` | Allows toggling of post match death animation.
`bolt.profile.pingfactor`     | Access to change queue ping range.
`bolt.profile.edit`           | Access to edit mode for building.
`bolt.profile.admin`          | Access to admin features (/spawn for now).
`bolt.profile.silent`         | Access to silent mode for spectating.

## Admin

Visit [Features/Admin](/Bolt/Features/Admin) for more information regarding profile permissions.

Permission                 | Description

`bolt.admin.refill`        | Access to /bolt refill.
`bolt.admin.forfeit`       | Access to terminate matches.
`bolt.admin.debug`         | Access to Bolt debug commands.
`bolt.admin.resetprofile"` | Access to reset a player's profile.
`bolt.holo.admin`          | Access to manage holograms.
`bolt.import.admin`        | Access to import from frost.
